Scroll Down

In this example I have used JScript to make the button visible after window.scroll passes the threshold value of 300px.

When, User clicks on the button it will move to the top with animation duration of 300ms

I have used meta tag with content="width=device-width, initial-scale=1.0" to make the page responsive
